# 05 · Ads

Only run this if you do not have organic traffic yet. Simple lead-magnet ad driving to `/custom-strategy`. Cheap, fast, trackable. Ten visual variants, ten copy variants, low budget until one combination proves itself. Then scale.

## What you get out

- 10 visual ad variants as static HTML files (1080×1080 + 1080×1350)
- 10 copy variants (primary text + headline + description)
- Launch settings, kill rule, scale rule

## Run it

1. Make sure `/custom-strategy` is live and a Meta Pixel is firing on every page
2. Open `prompts/05-ads-build.md`
3. Paste into Claude
4. Generate creatives + copy
5. Launch in Meta Ads Manager with the settings in `config.json`

## Watch out for

- **Ads are an accelerator, not a strategy.** If the first four steps are wrong, ads burn cash.
- Killing too early. Give each creative 100 clicks before deciding.
- Scaling on impressions, not conversions. The only metric that matters is cost per qualified lead.
