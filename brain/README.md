# 01 · Brain

Build the foundational knowledge base. Every other skill in this repo reads from these documents. Skip this and you fix the same bug ten times in the next four steps.

## What it produces

Twelve markdown documents in `brain/output/`:

- `BRAND.md`, `LOCKED_TRUTHS.md`, `PRODUCTS.md`, `CUSTOMERS.md`, `COMPETITORS.md`
- `KEYWORDS.md`, `FAQ_BANK.md`, `ABOUT.md`, `RESULTS.md`
- `GTM.md`, `OPERATIONS.md`, `ROADMAP.md`

## Run it

1. Fill out `../intake.json` with the business details
2. Open `prompts/01-brain-build.md`
3. Paste into Claude
4. Answer the 5 questions Claude asks
5. Output lands in `brain/output/`

## Watch out for

- Fabricated stats. Never invent a number. Mark `[VERIFY]` if unsure.
- Competitor copy bleeding in. Study them for structure, not language.
- Marketing jargon. B2-level English only.
